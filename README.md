# Tarot для Подруг

Проект: интерактивная колода Таро с персональными картами для подруг.  

## Возможности
- Вытянуть одну карту.
- Вытянуть три карты.
- Просмотр значений и персонажей карт.
- Простая визуализация карт.

## Структура проекта
- `index.html` — веб-страница.
- `style.css` — стили для страницы.
- `cards.js` — данные всех карт Таро.
- `script.js` — логика выбора карт и отображения.
- `cards/` — папка с изображениями карт (кубки, мечи, жезлы, пентакли, старшие арканы).
